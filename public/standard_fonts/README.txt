PDF.js Standard Fonts

This directory contains font files required by PDF.js to properly render PDFs.
The fonts should be extracted from the pdfjs-dist package during the build process.

Font files that should be copied here include:
- FoxitSans.pfb
- FoxitSansBold.pfb
- FoxitSansItalic.pfb
- FoxitSansBoldItalic.pfb
- FoxitSerif.pfb
- FoxitSerifBold.pfb
- FoxitSerifItalic.pfb
- FoxitSerifBoldItalic.pfb
- FoxitMonoBold.pfb
- FoxitMonoItalic.pfb
- FoxitMonoBoldItalic.pfb
- FoxitMono.pfb

These files should be copied automatically by the copy-pdf-fonts.js script.
